# Prisma
## A Nestjs project
### With some interesting `prisma` feature
- with dbml (copy your schema.dbml to https://dbdiagram.io/d, or use `erd` to know your dbml)
- with dto class definition
- with swagger model schemeas
- with docs
- with jsonSchema
- with erd

## run
``` js
yarn dev
```

## migrate
``` js
yarn migrate
```

## generate
``` js
yarn generate
```

## doc
``` js
yarn doc:serve
```

## erd
``` js
yarn erd:serve
```
